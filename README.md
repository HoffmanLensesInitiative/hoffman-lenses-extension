# Hoffman Lenses Initiative — Website

**hoffmanlenses.org**

The official website of the Hoffman Lenses Initiative — a human rights case for abolishing behavioral manipulation systems, and the home of the open-source Hoffman Lenses browser extension.

## About

The Hoffman Lenses Initiative is an independent, non-partisan, non-commercial project dedicated to making Behavioral Manipulation Systems visible, legally accountable, and ultimately obsolete.

Named for the glasses in John Carpenter's 1988 film *They Live* — which allowed the wearer to see the hidden messages embedded in ordinary reality.

## Deployment

This site is deployed via Cloudflare Pages. Every push to `main` triggers an automatic deployment to hoffmanlenses.org.

No build step required — pure static HTML/CSS/JS.
